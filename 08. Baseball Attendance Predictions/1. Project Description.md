<h1>Project Description </h1><br>
This project focused on understanding what drives attendance at Major League Baseball games by analyzing data from over 23,000 games between 2015 and 2025. The goal was to identify key factors that influence fan turnout and provide insights that teams could use to improve marketing and engagement strategies. <br>

I conducted exploratory data analysis on variables such as day of the week, game time, weather conditions, team performance, star players, and ballpark characteristics. Using these insights, I built a series of regression models using a stepwise approach, progressively adding variables and interaction terms to improve model performance. This included testing transformations, interaction effects, and seasonality components such as game number and lagged attendance trends. <br>

The final model achieved strong predictive performance, explaining approximately 73% of the variance in attendance. Key findings showed that factors like home team performance, star player presence, weekend scheduling, and iconic ballparks significantly increase attendance, while poor weather conditions and less popular venues reduce turnout. Interestingly, some expected relationships—such as late-season divisional games boosting attendance—were weaker than anticipated, highlighting the complexity of fan behavior. <br>

Overall, this project demonstrates my ability to work with large real-world datasets, build and evaluate predictive models, and translate statistical results into meaningful business insights. <br>
