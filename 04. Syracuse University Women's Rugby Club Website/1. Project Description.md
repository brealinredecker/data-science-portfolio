<h1>Project Description</h1>
<h3>URL: <a href="https://brealinredecker.github.io/ist263/project-7/">Live Site</a></h3>

This project involved designing and developing a fully functional website for the Syracuse University Women’s Rugby Team. The goal was to create a centralized platform to share team information, highlight players, and showcase performance through a clean and engaging user experience.<br>

The site includes multiple pages, such as a homepage, roster page, and leaderboard page, allowing users to explore player information and team performance in an organized and accessible way.<br>

I built the site using HTML and CSS, leveraging Flexbox to create responsive layouts that adapt across different screen sizes. A key focus was usability and structure, ensuring that content was clearly organized while maintaining a visually appealing design. I also incorporated accessibility best practices, including semantic HTML and alt text, to improve overall site quality.<br>

Throughout development, I resolved challenges related to layout alignment, responsiveness, and validation through iterative testing and debugging. I expanded beyond the base requirements by adding additional pages and features such as image galleries and player statistics, enhancing both functionality and user engagement.<br>

Overall, this project demonstrates my ability to build a complete front-end web application, apply responsive design principles, and translate real-world needs into an engaging digital experience.<br>

<h2>Key Results</h2>
- Built a fully responsive multi-page website optimized for desktop and mobile use <br>
- Improved user experience through structured navigation and clear content organization <br>
- Enhanced engagement with interactive features such as leaderboards and player profiles <br>
