<h1>Project Description</h1>
<h3>URL: https://brealinredecker.github.io/ist263/project-7/ </h3> <br>

This project involved designing and developing a fully functional website for the Syracuse University Women’s Rugby Team. The goal was to create a centralized platform to share team information, highlight players, and showcase performance through a clean and engaging user experience. The site includes multiple pages such as a homepage with general team information, a roster page with player details, and a leaderboard page highlighting top performers across key statistics. <br>

I built the site using HTML and CSS, leveraging Flexbox to create responsive layouts that adapt across different screen sizes. A major focus was on usability and structure, ensuring that content was organized clearly while maintaining a visually appealing design. I also incorporated accessibility considerations such as alt text, semantic HTML tags, and validation tools to improve overall site quality. <br>

Throughout the development process, I encountered challenges with layout alignment, responsiveness, and validation errors, which I resolved through iterative testing and debugging. I also expanded beyond the base requirements by adding additional pages and features like image galleries and player statistics, demonstrating initiative and creativity in the final product. <br>

Overall, this project highlights my ability to build a complete front-end web experience from scratch, apply responsive design principles, and translate real-world needs into a functional and engaging website.
